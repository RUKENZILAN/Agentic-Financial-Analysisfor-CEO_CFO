#  Antigravity Financial Dashboard & Multi-Agent System
7 Agent creates an agentic environment for financial evaluation and, upon completion of the analysis, generates a CEO and CFO dashboard as an index.html file. It shows worst-case and best-case scenarios and uses international matrices. Updated version of previous study. (https://github.com/RUKENZILAN/Agentic-Financial-Analysis)

This repository contains the architecture and UI components for a financial multi-agent system designed to standardize, analyze, and visualize corporate financial data. The system features a **CEO Agent** acting as the central orchestrator, delegating tasks to specialized sub-agents (Analyst, Auditor, Optimizer, Forecaster, DuPont) and compiling the results into a strategic Executive Summary.

## Architecture
- `ceo_agent.py`: Central orchestrator managing the workflow, standardizing raw Excel data, and generating the executive markdown summaries.
- `ui_ux_agent.py`: Handles formatting the JSON payloads into `dashboard_data.js` for local frontend rendering.
- `index.html`, `app.js`, `style.css`: A dynamic, modern, glassmorphism-styled web dashboard used to visualize financial metrics, charts, and strategic narratives.

## Recent Updates & Enhancements
- **Diplomatic & Balanced Tone**: Refined the CEO Agent's AI prompt to maintain a highly professional, moderate, and collaborative analytical tone.
- **Role Separation**: Executive Summaries are now explicitly divided into **CFO Perspectives** (risk, liquidity, debt) and **CEO Perspectives** (growth, market leadership).
- **Scenario Analysis**: Every assessment now includes explicit **Best-Case** and **Worst-Case** scenarios.
- **McKinsey & Eisenhower Formats**: Deep strategic assessment using the McKinsey 3-Horizons framework and prioritized action items via an Eisenhower Matrix.
- **Markdown UI Rendering**: Integrated `marked.js` directly into the dashboard to cleanly render the Markdown output generated by the AI agent.
- **Light/Dark Mode Toggle**: Added a 🌓 Theme button dynamically altering CSS variables and Chart.js properties for optimal viewing in any lighting.
- **Bilingual Support (EN/TR)**: Implemented a 🇬🇧 EN / 🇹🇷 TR language toggle directly on the dashboard that dynamically switches static UI labels, chart legends, and mock executive summaries without requiring a page refresh.
